Question Converter
=========================
Convert questions into a different format

Development
------------------------------


| Command            | Task                                                  |
| ------------------ | ---------------------------------------------------   |
| `yarn dev`         | Run development server.                               |
| `yarn build`       | Build production ready code into `dist/`.           |
| `yarn lint`        | Fix style of code in `src/`.                          |
| `yarn test`        | Run the unit tests via Mocha.                         |
